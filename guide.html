üöÄ Getting Started

What is Web Scraping?

Web scraping means using code to automatically read information from websites ‚Äî like copying text, links, or images from a page, but letting a script do it for you. Playwright is a tool that lets your code control a real web browser (Chrome, Firefox, etc.).

Quick Start (Windows)

1

2

Wait for Install

It will download everything you need (takes a few minutes)

3

Run your first scraper and watch the magic happen

Manual Installation

Step 1: Make sure Node.js is installed

node --version

If you don't see a version number, download Node.js from nodejs.org

Step 2: Install dependencies

npm install

Step 3: Install Playwright browsers

npx playwright install chromium
First time? Start with the beginner script: npm run beginner

üîç How Web Scraping Works

The Basic Pattern

// 1. Open a browser
const browser = await chromium.launch();

// 2. Open a new page (like a new tab)
const page = await browser.newPage();

// 3. Go to a website
await page.goto("https://example.com");

// 4. Find and read data
const data = await page.textContent("h1");

// 5. Close the browser
await browser.close();

Finding Elements (CSS Selectors)

To scrape something, you need to tell Playwright where to find it using CSS selectors:

  • .classname ‚Äî finds elements with that class
  • #id ‚Äî finds the element with that ID
  • h1 ‚Äî finds all <h1> tags
  • div.container ‚Äî finds <div> with class "container"
  • a[href] ‚Äî finds all links that have an href attribute
Right-click any element on a webpage and choose "Inspect" to see its HTML and find the right selector!

üéØ Useful Playwright Tricks

Navigation

// Go to a page
await page.goto("https://example.com");

// Click a button
await page.click("button.submit");

// Type into an input
await page.fill("input#search", "hello world");

// Press Enter
await page.press("input#search", "Enter");

Reading Data

// Get text from one element
const title = await page.textContent("h1");

// Get text from multiple elements
const items = await page.$$eval("li", 
  elements => elements.map(el => el.textContent)
);

// Get an attribute (like href from a link)
const url = await page.getAttribute("a.link", "href");

Waiting

// Wait for an element to appear
await page.waitForSelector(".results");

// Wait for the page to load
await page.waitForLoadState("domcontentloaded");

// Wait a specific amount of time (milliseconds)
await page.waitForTimeout(3000); // 3 seconds

Screenshots & PDFs

// Take a screenshot
await page.screenshot({ path: "screenshot.png" });

// Take a full-page screenshot
await page.screenshot({ 
  path: "full-page.png", 
  fullPage: true 
});

// Save as PDF
await page.pdf({ path: "page.pdf" });
Be respectful when scraping! Don't send too many requests too quickly. Add delays between requests to avoid overloading servers.

üö® Troubleshooting

Problem: "node is not recognized" or "npm is not recognized"
Solution: Node.js isn't installed. Download it from nodejs.org and make sure to check "Add to PATH" during installation.
Problem: "Cannot find module 'playwright'"
Solution: Run npm install in the project folder.
Problem: "Browser not found" or "Executable doesn't exist"
Solution: Run npx playwright install chromium
Problem: The scraper can't find any elements (returns empty array)
Solution: Your CSS selector might be wrong, or the page hasn't loaded yet. Try:
  • Right-click the element and choose "Inspect" to verify the selector
  • Add a wait: await page.waitForSelector(".your-class")
  • Give the page more time: await page.waitForTimeout(3000)
Problem: "Navigation timeout" or page didn't load
Solution: The website might be slow. Try increasing the timeout:
await page.goto(url, { timeout: 60000 })
Problem: Browser window closes immediately
Solution: Add a delay before closing: await page.waitForTimeout(5000) right before browser.close()
Stuck? Try the ultra-beginner script first (npm run beginner) to make sure everything is set up correctly!
